# Coworking Frontend - Vue 3

Полнофункциональное фронтенд приложение для системы управления коворкингом.

## 🚀 Особенности

✅ **Авторизация и регистрация** - JWT токены  
✅ **Две роли** - Администратор и обычный пользователь  
✅ **Панель пользователя** - Просмотр и управление своими бронированиями  
✅ **Админ-панель** - Управление всеми бронированиями, статистика  
✅ **Интеграция с Backend** - Полное взаимодействие с API на Express  

## 📋 Требования

- Node.js 14+
- npm 6+
- Backend запущен на `http://localhost:3000`

## 📦 Установка

### 1. Установить зависимости

## 🎯 Функциональность

### Для пользователя:

1. **Авторизация / Регистрация** - Создание аккаунта или вход
2. **Просмотр бронирований** - Список своих забронированных рабочих мест
3. **Управление бронированиями** - Отмена забронированных мест
4. **Статусы** - Просмотр статуса каждого бронирования (pending, confirmed, completed, cancelled)

### Для администратора:

1. **Управление бронированиями** - Просмотр всех бронирований, подтверждение, отмена
2. **Отправка напоминаний** - Отправка email напоминаний пользователям
3. **Статистика** - Просмотр количества бронирований по статусам
4. **Система** - Информация о версии и статусе API

## 🔌 API интеграция

Все API запросы делаются через Axios с автоматической передачей JWT токена.

### Основные endpoints:

- `POST /login` - Вход
- `POST /register` - Регистрация  
- `GET /get/booking` - Получить бронирования
- `POST /post/booking` - Создать бронирование
- `PATCH /cancel/booking/:id` - Отменить бронирование
- `PATCH /confirm/booking/:id` - Подтвердить бронирование (админ)
- `POST /remind/booking/:id` - Отправить напоминание (админ)

## 📞 Проблемы

### "Не могу залогиниться"
- Убедитесь что Backend запущен на порту 3000
- Проверьте правильность email и пароля
- Посмотрите консоль браузера (F12) для подробных ошибок

### "404 Not Found"
- Проверьте что вы на правильном URL (`http://localhost:5173`)
- Очистите браузер кэш (Ctrl+Shift+Delete)

### "Network Error"
- Убедитесь что Backend запущен: `cd /Users/getaway/coworking && npm run dev`
- Проверьте что вы используете правильный API URL в api.js
